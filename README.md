Brazilian E-Commerce Analysis (Power BI)
📌 Overview

This project analyzes the Brazilian e-commerce dataset (Olist) using Power BI to uncover actionable insights into sales performance, customer behavior, and business trends.

The goal is to transform raw transactional data into an interactive dashboard that helps stakeholders understand performance drivers and identify opportunities for growth.

🎯 Objectives

Analyze overall sales performance and trends

Understand customer purchasing behavior

Evaluate regional performance

Identify opportunities to improve customer retention

🔍 Key Insights

Identified a low repeat customer rate, indicating weak customer loyalty

Highlighted regional variations in sales performance

Revealed seasonal trends in order volume

Provided insights for improving retention and long-term customer value
